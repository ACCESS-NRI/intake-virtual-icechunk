# Installation

**Using pixi:**

```bash
pixi add intake-virtual-icechunk
```

**Using pip:**

```bash
python -m pip install intake-virtual-icechunk
```

**From source:**

```bash
git clone git@github.com:ACCESS-NRI/intake-virtual-icechunk.git
cd intake-virtual-icechunk
python -m pip install -e .
```
